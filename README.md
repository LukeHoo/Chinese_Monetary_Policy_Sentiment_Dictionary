**词典构建**

本词典是在姜富伟、孟令超和唐国豪构造的中文金融情感词典基础上，针对央行沟通文本的特点，在仔细阅读央行货币政策报告的文本后，参照冀志斌和宋清华（2012）、林建浩和赵文庆（2015）等以往文献关于选词的设定，添加了适用于央行沟通文本的一些常用情感词而成。本词典适用于计算人民银行《货币政策执行报告》等央行中文书面沟通文本，词典相关信息和情感词举例如下：
